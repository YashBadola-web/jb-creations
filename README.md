# JB Creations

A beautiful, responsive e-commerce application for artisanal crafts.

## Project Info

**URL**: https://jb-creations.vercel.app/

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide React

## Features

- **Storefront**: Browse products by category (Resin Art, Kids, Decor).
- **Cart & Checkout**: Full shopping cart experience with simulated checkout (UPI, COD).
- **Admin Panel**: Manage orders (protected by login).
- **About Us**: Story and contact form.
